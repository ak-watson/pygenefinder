Background
----------

This is a Python package for detecting antibiotic resistance (AMR) genes in nucleotide sequences.
It uses Blast to find hits to known AMR gene sequences. The inputs are fasta files.

Dependencies
------------

numpy
pandas
biopython
matplotlib
pandastable
future

Installation
------------

  pip install pyamrfinder

Usage
-----

  pyamrfinder -p <path-to-fasta-files> -d <database>
